纯文字hexo博客搭建教程

发表于2023-01-10|更新于2023-01-14|文字天下
|阅读量:

​ 关于Hexo博客搭建，一步一个脚印慢慢来，不要因为一个不在意，换来后面东寻西问都难以解决的问题，那就得不偿失了。别放弃，问题实在解决不了大不了重头再来嘛~没准就可以顺利成功了呢！

​ 首先，搭建hexo博客我们要祭出两大法宝——一个git，一个node.js，快去官网下载吧！至于为什么要下载他们,嗯…我想：应该是他们上辈子积德深厚吧！下载好了嘛？那就开始吧！踏向成功第一步：请点击win+R，输入cmd，点个确定试试看啦！踏向成功第二步：请输入命令—1、git -v,2、node -v，什么？不对？！我早有准备，请点开你的设置，搜索高级系统设置，打开继续走—>环境变量—>path(上面一个，下面一个)—>编辑—>新建。复制你的git文件路径，粘贴到你的新建上面(上下均贴)，node同上。再次输入命令，看见版本号了吧？恭喜你，离成功又近了一步！！！

​ 接下来，就是hexo的舞台了！在此之前，我想请你在非C盘处新建一个名为myblog的文件夹。为什么非C？当然是为你，啊不，为你的电脑着想了！在此文件夹内点击右键，寻找一下git bash here，打开后输入命令—npm install -g hexo-cil。安装好了就接着输入hexo init 进行hexo初始化。去看看文件夹里面是不是多了好多文件？惊不惊喜，意不意外？！接着我们继续输入命令hexo -s，查看能否启动成功，是不是有个http开头的网址？去浏览器打开看看，一个以hexo为大标题的页面展现在你的眼前，Hello World作为赠品就免费送你了！恭喜你，离起点又远了一步！！！

​ 别急，还没结束呢，页面你是能看到了，得让别人看到嘞。这时候，第三件法宝横空出世！没错，他就是人见人爱，花见花开，车见车爆胎的-Github！！！接下来，还得麻烦你去Github官网创建你的世界，英文看不懂？！请前往英语词典进行查找，网易有道你也值得拥有。踏向成功第六步：登录后把箭头放在你的右上角头像处，点开你的仓库，点击new新建一个仓库吧！仓库名请设置为：GitHub用户名+.github.io (谁要是把加号也输进去,给我发私信,我保证不笑)，请为下面的Add a README file拉勾，点击Create repository。踏向成功第七步：生成ssh添加到Github—打开git bash here—>输入命令git config –global user.name “yourname”(yourname是你的GitHub用户名,不要整错了嘞)，什么都没显示，紧接着输入git config –global user.email “your github email”(也就是你创建GitHub时的邮箱)，还是什么也没有？！再一再二不再三了哈!紧接着输入命令ssh-keygen -t rsa -C “your github email”，大写小写分清楚哦~连敲三次回车(Enter)。打开文件查找.ssh，发现有一个公钥，一个私钥，id_rsa.pub是公钥，打开他复制里面所有内容。进入GitHub，箭头放在右上角头像上，进入下面的setting，点击ssh and GPG keys—>New SSH key。title您随意，Key处是要粘贴公钥嘀!点击SSH key。恭喜你，离中点已经有些距离了！！！

​ 最后！激动不？！请打开你的myblog文件夹下的_config.yml文件，如果你电脑上没有打开这类文件的工具且不想下载的话，请将该文件改为txt文件，记事本打开！翻到最后一点，看到deploy，欧克欧克，进行修改，修改如下：注意！！！冒号后面有一个空格,格式万不可出错！

deploy:

type: git

repo: 你的GitHub仓库的SSH链接

branch: main 接下来，我们需要安装一个插件，在myblog文件夹中打开git bash here,输入命令：npm install hexo-deployer-git –save。依次执行以下命令hexo -c;hexo -g;hexo -d。(其中，hexo -c可使用hexo -clean,hexo -d可使用hexo -deploy)。注意：最后一条命令(hexo -d)最后会让输入一个yes，不要忘了哈!，回到GitHub上打开仓库，看到各种各样的文件，表示部署成功，此时你的博客链接为：https://GitHub用户名+.github.io,打开.恭喜你，走向成功！！！

​ 是不是对页面不太感兴趣？莫慌莫慌，俺也一样！！！明天教你，嘻嘻~加油！！！如果过程中有问题，欢迎加我qq进行询问！可是我是小白诶,我也不可能都会，如果我也解决不了，请你不要着急！！！待我去找学长聊聊天
